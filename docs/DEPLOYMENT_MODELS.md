# Deployment Models

## Private Cloud

Dedicated enterprise cloud environments powered by OpenStack.

Characteristics:
- Infrastructure ownership
- Enterprise governance
- Controlled environments
- Security and compliance alignment

## Edge Cloud

Distributed edge-native infrastructure powered by MicroCloud.

Characteristics:
- Small-footprint deployment
- Distributed workloads
- Low-latency operations
- Regional delivery

## Hybrid Cloud

Hybrid infrastructure orchestration powered by Apache CloudStack.

Characteristics:
- Multi-environment management
- Cross-cloud orchestration
- Workload portability
- Enterprise integration

## Composable Cloud

A modular operational architecture where services are assembled based on enterprise requirements.

Characteristics:
- Modular service composition
- API-first integration
- Infrastructure extensibility
- Independent lifecycle management
