# Unboxd Cloud Service Catalog

This catalog describes the enterprise service domains that make up the Unboxd Cloud platform.

## Cloud Foundation

| Service | Role |
|---|---|
| OpenStack | Private cloud infrastructure |
| MicroCloud | Edge cloud infrastructure |
| Apache CloudStack | Hybrid cloud orchestration |
| S3-Compatible Storage | Object storage foundation |

## Cloud Native Operations

| Service | Role |
|---|---|
| Podman | Container lifecycle management |
| Harbor | Container image registry |
| Apache APISIX | API gateway and traffic management |

## Identity & Communication

| Service | Role |
|---|---|
| Keycloak | Enterprise IAM |
| Logto | Application SSO |
| Stalwart | Mail infrastructure |
| Novu | Notification management |

## Observability & Reliability

| Service | Role |
|---|---|
| SigNoz | Logs, metrics, and traces |
| GlitchTip | Error tracking and uptime monitoring |

## Application Services

| Service | Role |
|---|---|
| WordPress | Agentic CMS |
| WooCommerce | Agentic commerce |
| Ghost | Publishing |
| Postiz | Social publishing |

## Agentic Operations

Each service domain can be paired with an operational agent responsible for provisioning, configuration, monitoring, remediation, documentation, and lifecycle management.
