# Security and Governance

## Overview

Unboxd Cloud is designed for enterprise environments where governance, operational visibility, access control, and infrastructure ownership are required.

## Identity and Access

Identity and access management are delivered through:

- Keycloak
- Logto
- SSO integration
- Role-based access control
- Federated identity integration

## Operational Governance

Operational governance is supported through:

- Observability pipelines
- Logs, metrics, and traces
- API gateway governance
- Service-domain operational agents
- Monitoring and remediation workflows

## Deployment Governance

Supported deployment governance models:

- Private cloud ownership
- Hybrid cloud governance
- Edge infrastructure governance
- Multi-environment operations

## Security Principles

- Enterprise IAM integration
- API-first control layers
- Infrastructure segmentation
- Observability-driven operations
- Operational traceability
- Open-source infrastructure transparency

## Operational Visibility

Operational visibility is delivered through:

- SigNoz observability
- GlitchTip error tracking
- Infrastructure monitoring
- Service health visibility
- Distributed operational tracing
